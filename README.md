# GatherWise AI Moodboard

A collaborative wedding moodboard built with the GatherWise design system — PowerPoint-style slide editor with drag-and-drop images and text, sections sidebar, and fixed comment panel.

## Run locally

```bash
npm install
npm run dev
```

Open the URL shown in the terminal (typically `http://localhost:5173`).

## Features

- **Slide canvas editor** — each section has multiple 16:9 slides with freeform drag/resize
- **Image & text elements** — add, move, resize, and double-click text to edit inline
- **Multi-slide decks** — add, duplicate, and delete slides (planner only)
- **Vision brief** — AI-generated aesthetic summary (mock "Summarize vision" action)
- **Comment threads** — select an image on the canvas to open its thread panel
- **AI suggestions** — pin suggested images via the Add image drawer
- **Role toggle** — switch between Planner and Client views (demo control at bottom)

## Stack

React 19 · TypeScript · Vite · react-rnd · Lucide icons · GatherWise design tokens
